<img />

<figure>

</figure>

Tag que deve ser precedida pela tag figure, é utilizada para inserir imagens no html.

Após a tag img, é possível utilizar a tag figcaption para colocar uma legenda.

Atributos:

src - Atributo para colocar o caminho da imagem.

alt - Texto alternativo para caso a imagem não carregue ou tenha que ser descrita por um browser, o atributo alt descreve a imagem.

longdesc - link para uma descrição mais detalhada da imagem. Os valores possíveis são um URL ou um elemento id.

title Utilizar o atributo title para a imagem listar em mecanismos de busca.

Também pode-se utilizar a tag object.

<object data="imagem.jpg" width="300" height="200">

Formulários

  • Utilizar o atributo tabindex para colocar uma ordem de preenchimento dos inputs caso o usuário utilize a tecla tab.
  • Utilizar a tag fieldset e a legend para agrupar elementos relacionados em um formulário.
  • Para elementos relacionados no formulário, utiliza-se a tag legend
  • Utilizar a tag label para identificar o campo.
  • Pode ser usado o atributo autocomplete="on" ou "off" para input ou form.
  • Utilizar o autofocus em: button, input, select e textarea.

    Exemplo: button type="button" autofocus>Click Me! /button>

  • Utilizar o atributo enctype="multipart/form-data" para que os dados sejam enviados codificados. Apenas pelo method="post".

  • formaction

    Substitui o action="get" ou "post". Utiliza-se no button type submit

    button type="submit" formaction="/action_page2.php">Submit to another page /button>

Atributos globais

itemscope

Define o escopo dos metadados associados. Ele cria um novo item que segue algum padrão itemtype, que pode ser o padrão do schema.org.

Se não for relacionado a um itemtype deve ser associado a um itemref.

Após o itemscope, coloca-se os itemprop que estão definidos pela comunidade do itemtype.

Deve-se procurar no site schema.org todos os atributos itemprop para cada item criado pelo itemscope.

Exemplo:

<div itemscope itemtype ="http://schema.org/Movie">

<h1 itemprop="name">Avatar</h1>

<span>Director: <span itemprop="director">James Cameron</span> (born August 16, 1954)</span>

<span itemprop="genre">Science fiction</span>

a href="../movies/avatar-theatrical-trailer.html" itemprop="trailer">Trailer</a>

</div>

Ver mais exemplos em:

itemtype

O itemtype é usado para especificar a URL válida de um vocábulo (como schema.org), que descreve o item e seu contexto de propriedades.

itemprop

Nos itemprop estão os dados associados ao itemscope, que devem estar de acordo com algum itemtype, como por exemplo o schema.org.

Exemplo:

<div itemscope itemtype ="http://schema.org/Movie">

<h1 itemprop="name">Avatar</h1>

<span>Director: <span itemprop="director">James Cameron</span> (born August 16, 1954)</span>

<span itemprop="genre">Science fiction</span>

a href="../movies/avatar-theatrical-trailer.html" itemprop="trailer">Trailer</a>

</div>
